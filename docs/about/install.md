# Quick Install

Install Node.js and create a new KnowledgeBase site:

```
npx knowledgebasetech@latest init my-website
```

Start the site:

```
cd my-website
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and checkout the app.
